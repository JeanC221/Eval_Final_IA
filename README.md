## Predicción de Diabetes usando Aprendizaje Supervisado y No Supervisado

El objetivo es predecir la presencia de diabetes a partir de variables clínicas utilizando técnicas de:

-  Aprendizaje 
-  Aprendizaje 
-  Reducción de dimensionalidad (PCA, t-SNE)  
-  Implementación manual de un algoritmo 

El proyecto incluye un pipeline completo desde la exploración inicial hasta la implementación algorítmica.

---

# Descripción del Dataset

- Nombre: Diabetes Prediction Dataset
- Tamaño: 100,000 registros
- Variables: edad, IMC, hipertensión, glucosa, HbA1c, tabaquismo, etc.
- Variable objetivo: `diabetes`
- Características:
  - Completamente tabular
  - Sin valores faltantes
  - Excelentes variables predictoras clínicas
